## JavaScript Array Methods Practice

Этот репозиторий содержит решения задач на различные методы работы с массивами в JavaScript. Каждая задача использует один или несколько методов для обработки данных и достижения определенной цели.
Каждый метод находится на своей ветке.

### Использование map:

Задача: Умножить каждое число в массиве на 2.

Пример: [1, 2, 3, 4] -> [2, 4, 6, 8]

### Использование filter:

Задача: Отфильтровать строки, длина которых больше 5 символов.

Пример: ['apple', 'banana', 'kiwi', 'grapefruit'] -> ['banana', 'grapefruit']

### Использование reduce:

Задача: Найти сумму всех чисел в массиве.

Пример: [1, 2, 3, 4] -> 10

### Использование forEach:

Задача: Вывести в консоль имена всех студентов из массива объектов.

Пример: [{name: 'Alice', age: 21}, {name: 'Bob', age: 22}] -> Alice, Bob

### Использование find:

Задача: Найти книгу по заданному названию в массиве объектов.

Пример: [{title: '1984', author: 'George Orwell'}, {title: 'To Kill a Mockingbird', author: 'Harper Lee'}] -> {title: '1984', author: 'George Orwell'}

### Использование some:

Задача: Проверить, содержит ли массив хотя бы одно четное число.

Пример: [1, 3, 5, 7] -> false

### Использование every:

Задача: Проверить, все ли числа в массиве положительные.

Пример: [1, 2, 3, 4] -> true


### Анализ данных о продажах:

Задача: Рассчитать общую сумму продаж для каждого товара, используя map и reduce.

Пример: [{date: '2023-10-01', item: 'apple', amount: 100}, {date: '2023-10-02', item: 'banana', amount: 150}, ...]


### Фильтрация данных о пользователях:

Задача: Получить список пользователей с активной подпиской и возрастом старше 30 лет, используя filter.

Пример: [{name: 'Alice', age: 28, isSubscribed: true}, {name: 'Bob', age: 35, isSubscribed: false}, ...]


### Задача: Вывести в консоль все задачи с высоким приоритетом, которые еще не выполнены, используя forEach.

Пример: [{task: 'Write report', status: 'completed', priority: 'high'}, {task: 'Send email', status: 'pending', priority: 'low'}, ...]

### Динамическое обновление списка задач (To-Do List):

 Задача: Реализовать функционал, который будет обновлять список задач на странице в реальном времени, используя JavaScript и взаимодействие с DOM. Пользователи могут добавлять новые задачи, а также сортировать задачи через интерфейс.
